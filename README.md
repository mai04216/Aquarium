# Aquarium

放置系×箱庭系の水槽育成アプリ「マイアクアリウム(仮称)」。

## 概要

最初は水しか入っていない水槽に魚・海藻・岩を配置し、魚が生成するコインでさらに水槽を拡張していくアプリです。詳細は設計書を参照してください。

## 現在の実装状況

- フェーズ1(一部): 水槽の表示と、魚1匹のCSSアニメーションによる遊泳表現

## セットアップ・起動方法

ビルドツールは使用していません。`index.html` をブラウザで直接開くか、簡易サーバーで配信してください。

```bash
# 例: Python の簡易サーバーを使う場合
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 技術スタック

- HTML / CSS / JavaScript(素の実装、ビルドツールなし)
- アニメーション: CSS `@keyframes`
