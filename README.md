# Aquarium

放置系×箱庭系の水槽育成アプリ「マイアクアリウム(仮称)」。

## 概要

最初は水しか入っていない水槽に魚・海藻・岩を配置し、魚が生成するコインでさらに水槽を拡張していくアプリです。詳細は設計書を参照してください。

## 現在の実装状況

- 水槽の表示、魚の遊泳・海藻の揺れ・泡演出
- コイン自動生成/残高表示
- ショップ(魚・装飾・水槽)、インベントリ、配置・移動・撤去
- 複数水槽の保持・切り替え
- セーブ/ロード(localStorage)
- BGM・効果音、設定画面(サウンドON/OFF)

## セットアップ・起動方法

ビルドツールは使用していません。マスタデータを `fetch` で読み込むため、`index.html` は**簡易サーバー経由**で開いてください(`file://` で直接開くと読み込みに失敗します)。

```bash
# 例: Python の簡易サーバーを使う場合
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

## データ調整(経済バランス)

魚・装飾・水槽のマスタデータは `data/master.json` に外部化しています。価格・生成間隔・生成量・配置上限などは、このJSONを編集するだけで調整できます(コードの変更は不要)。

## 技術スタック

- HTML / CSS / JavaScript(素の実装、ビルドツールなし)
- アニメーション: CSS `@keyframes`
- サウンド: WebAudio API(手続き生成)
- マスタデータ: JSON(`data/master.json`)
